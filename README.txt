This plugin activates the tutorial section of the INDI Websitesystem. 

== Description ==

INDI Websitesystem users get a tutorial overview of all videos using the INDI Websitesystem.
This plugin works only with the following User-Profiles:

*   Administrator
*   INDI-User
*   INDI-Blogger
*   INDI-Editor
*   INDI-Manager
*   INDI-Admin

== Installation ==

This plugin is already installed on the INDI Websitesystem.
